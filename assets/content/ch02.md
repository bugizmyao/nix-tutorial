# Nixをインストールする

Nixには2つのインストーラがあります。

| インストーラ | 提供元 | 特徴 |
|-------------|--------|------|
| **公式** | NixOS Foundation | 歴史が長いが、アンインストールが面倒 |
| **Determinate Nix Installer**（推奨） | Determinate Systems | クリーンにアンインストール可、Flakes標準有効 |

このチュートリアルでは **Determinate Nix Installer** を使います。

---

# Determinate Nix Installer を推奨する理由

- ✅ **Flakes と `nix` コマンドがデフォルトで有効** — 後で手動有効化しなくていい
- ✅ **アンインストールがコマンド一発** — 気軽に試せる
- ✅ **macOSアップデートに強い** — Apple のシステムボリューム再構築で壊れにくい
- ✅ **インストールが早い** — 1〜2分で終わる

> [note] 公式インストーラを既に使っている場合は、無理に切り替える必要はありません。

---

# インストールコマンド

ターミナルで以下を実行します。

```bash
curl --proto '=https' --tlsv1.2 -sSf -L \
  https://install.determinate.systems/nix | \
  sh -s -- install
```

途中で対話的に確認が出るので、基本は `Y` で進めればOKです。

> [hint] 5〜10分でインストール完了。途中で sudo パスワードを聞かれることがあります。

---

# 動作確認

インストール後、**新しいターミナルウィンドウを開いて** 以下を試します。

```bash
$ nix --version
nix (Determinate Nix 3.x.x) 2.x.x
```

```bash
$ nix run nixpkgs#hello
Hello, world!
```

`Hello, world!` が出れば成功です。`nix run` は「インストールせずに一度だけ実行」する魔法のコマンド。

# 演習 — 自分で叩いてみる

> [try] 以下の3つを叩いてみてください。
> ```bash
> nix --version
> nix run nixpkgs#hello
> nix run nixpkgs#cowsay -- "Hello Nix"
> ```
> `cowsay` で牛が喋ったら成功です。インストールはしていません — Nix が一時的に環境を作って実行しただけ。
