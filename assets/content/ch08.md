# nix-darwin とは

`nix-darwin` は、**macOSのシステム全体** を Nix で宣言的に管理するツールです。

- システムパッケージ
- Homebrew連携（App Store以外のGUIアプリも）
- macOS の各種設定（`defaults` に相当）
- フォント、サービス、ユーザー

NixOS の macOS版、と思えば近いです。

> [note] System Integrity Protection（SIP）の制約上、OSの全てを置き換えるわけではありません。「ユーザー設定可能な範囲をNix化する」ツールです。

---

# 何が嬉しいのか

- **Mac新調時の再構築が瞬時に終わる** — `darwin-rebuild switch` 1発で元通り
- **「これどうやって設定したっけ？」が消える** — 全部コードに残る
- **ロールバック可能** — 設定変更で何かおかしくなっても1個前に戻れる
- **同期** — 仕事Mac / 個人Mac で設定を共有できる

> [hint] Mac新調時に泣いたことがある人ほど刺さるツールです。

---

# インストール（Flake版）

最初に設定ディレクトリを作ります。

```bash
$ mkdir -p ~/.config/darwin && cd ~/.config/darwin
$ nix flake init -t nix-darwin
```

雛形が生成されるので、`hostname` を自分のMacに合わせて編集します。

```bash
$ scutil --get LocalHostName     # ← 自分のhostnameを確認
$ vim flake.nix                  # 雛形の hostname を書き換え
```

最初の適用:

```bash
$ nix run nix-darwin -- switch --flake ~/.config/darwin
```

成功すると以後は `darwin-rebuild switch --flake ~/.config/darwin` で適用できるようになります。

---

# configuration の基本構造

`~/.config/darwin/flake.nix` の中で参照される設定ファイル（典型的には `configuration.nix`）はこんな形:

```nix
{ pkgs, ... }:
{
  # システム全体で使うパッケージ
  environment.systemPackages = with pkgs; [
    git
    jq
    ripgrep
    fd
  ];

  # 必要な機能を有効化
  nix.settings.experimental-features = [ "nix-command" "flakes" ];

  # システムバージョン（最初に決めたら変えない）
  system.stateVersion = 4;
}
```

# Homebrew 連携

GUIアプリは Nix にないものが多いので、Homebrew Casks を nix-darwin から呼びます。

```nix
{
  homebrew = {
    enable = true;
    onActivation = {
      autoUpdate = true;
      cleanup = "zap";    # 管理外を全削除
    };
    casks = [
      "google-chrome"
      "raycast"
      "rectangle"
      "1password"
    ];
    masApps = {
      "Xcode" = 497799835;
    };
  };
}
```

`darwin-rebuild switch` 一発で、宣言したものだけが残る状態になります。

> [hint] `cleanup = "zap"` は強力。管理外の cask を消すので注意。最初は `"uninstall"` から始めるのが無難。

# macOS 設定（defaults相当）

「Finderで拡張子を常に表示」「Dockを自動隠す」など、Mac設定がコードで書けます。

```nix
{
  system.defaults = {
    dock = {
      autohide = true;
      orientation = "left";
      tilesize = 48;
    };
    finder = {
      AppleShowAllExtensions = true;
      ShowPathbar = true;
      FXEnableExtensionChangeWarning = false;
    };
    NSGlobalDomain = {
      AppleInterfaceStyle = "Dark";
      KeyRepeat = 2;
      InitialKeyRepeat = 15;
    };
    trackpad = {
      Clicking = true;
      TrackpadThreeFingerDrag = true;
    };
  };
}
```

> [note] 設定可能なキー一覧: https://daiderd.com/nix-darwin/manual/index.html

# 適用と確認

設定を変更したら適用:

```bash
$ darwin-rebuild switch --flake ~/.config/darwin
building the system configuration...
applying ...
✓ Setup successful
```

失敗時は前の世代に自動的に留まるので、Macが壊れる心配はありません。

確認:

```bash
$ which jq          # → /run/current-system/sw/bin/jq
$ defaults read com.apple.dock autohide   # → 1
```

# 世代管理 — ロールバック

```bash
$ darwin-rebuild --list-generations
  10  2026-05-10 14:00:00
  11  2026-05-12 09:30:00
  12  2026-05-14 14:33:00   (current)
```

1つ前に戻す:

```bash
$ darwin-rebuild --switch-generation 11
```

> [hint] 「設定を変えたら何かおかしくなった」というときに即座に戻せる安心感。これがNixの真骨頂。
