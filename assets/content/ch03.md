# 覚えるべき7つのコマンド

Nixの日常使いは、まずこの7つを押さえれば十分です。

| コマンド | 用途 |
|---------|------|
| `nix search` | パッケージを検索 |
| `nix run` | 一回限り実行（インストールしない） |
| `nix shell` | 一時的にPATHに追加（subshell） |
| `nix profile install` | 永続インストール（Homebrew的） |
| `nix profile list` | インストール済み一覧 |
| `nix profile remove` | アンインストール |
| `nix store gc` | 不要なパッケージを削除 |

順番に見ていきましょう。

---

# nix search — 検索する

`nixpkgs` から欲しいパッケージを探します。

```bash
$ nix search nixpkgs ripgrep
* legacyPackages.aarch64-darwin.ripgrep (14.1.0)
  A utility that combines the usability of The Silver Searcher with the raw speed of grep
```

> [hint] Web版もあります: https://search.nixos.org/packages
> ブラウザでざっと探すならこちらが速いです。

---

# nix run — 一度だけ実行する

「ちょっと試したいだけ」のときに最高です。

```bash
$ nix run nixpkgs#cowsay -- "Hello"
 _______
< Hello >
 -------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```

`--` のあとは cowsay 自体への引数です。**実行が終わったらPATHから消えます**。

---

# nix shell — 一時的に環境に入れる

`nix run` は1回限りですが、`nix shell` は **subshell の間ずっと使える** モードです。

```bash
$ nix shell nixpkgs#nodejs_20
$ node --version
v20.x.x
$ exit       # ← 抜ける
$ node --version
zsh: command not found: node
```

複数パッケージも入れられます:

```bash
$ nix shell nixpkgs#nodejs_20 nixpkgs#yarn
```

---

# nix profile install — 永続インストール

Homebrew の `brew install` に近いコマンド。

```bash
$ nix profile install nixpkgs#jq
```

これで `jq` がPATHに常駐します。

> [note] あとで `home-manager` を使うようになると、ここは宣言的に書くようになります。
> `nix profile install` は手軽なお試し用と考えてOKです。

---

# nix profile list / remove

```bash
$ nix profile list
Name:               jq
Flake attribute:    legacyPackages.aarch64-darwin.jq
Store paths:        /nix/store/...-jq-1.7.1
```

削除は名前を指定:

```bash
$ nix profile remove jq
```

---

# nix store gc — ガベージコレクション

Nixは過去のバージョンも `/nix/store` に保管するので、放っておくと容量を食います。定期的に掃除しましょう。

```bash
$ nix store gc
```

30日以上前の世代だけ消したい場合:

```bash
$ nix profile wipe-history --older-than 30d
$ nix store gc
```

> [hint] 月1回くらい `nix store gc` を叩く習慣をつけると安心。

---

# nixpkgs って何？

これまで何度も出てきた `nixpkgs#xxx` の正体は **nixpkgs リポジトリ** です。

- GitHub: https://github.com/NixOS/nixpkgs
- 8万以上のパッケージが定義されている世界最大級のパッケージリポジトリ
- 検索: https://search.nixos.org/packages

`nixpkgs#hello` は「nixpkgs リポジトリの hello パッケージ」という意味です。

# 演習 — コマンドに慣れる

> [try] 以下を順番に叩いて、コマンドの違いを体感しましょう。
> ```bash
> # 1. 検索
> nix search nixpkgs jq
>
> # 2. 一度だけ実行
> echo '{"a": 1}' | nix run nixpkgs#jq -- .
>
> # 3. 永続インストール
> nix profile install nixpkgs#jq
>
> # 4. 確認
> nix profile list
>
> # 5. 削除
> nix profile remove jq
> ```
> 5番でちゃんと消えるところまで確認できれば完璧です。
